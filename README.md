# Telecom-Churn-Prediction
Réalisation d’un projet end-to-end de Machine Learning pour la prédiction du churn télécom : nettoyage et préparation des données (Pandas, scikit-learn), entraînement de modèles (XGBoost, Random Forest, Logistic Regression), sauvegarde du pipeline (joblib) et création d’une API de prédiction avec FastAPI déployée via Uvicorn.
